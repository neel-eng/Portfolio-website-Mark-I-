Personal Portfolio Website

I built this portfolio website while learning  This project started asand experimenting about web design, a small practice exercise to try building a full webpage.
While working on this site, I learned how to structure a long single-page layouts and how different sections of a webpage fit together. I spent a lot of time experimenting with CSS Flexbox to align elements and control spacing, especially for sections with multiple columns.
One of the main things I explored in this project was CSS scroll animations. I experimented with keyframes and scroll-based animations to make images and elements move as the user scrolls. This helped me understand how positioning, transforms, and layering using z-index work in real projects.
At the moment, this website is not responsive and works best on desktop screens. Responsiveness is something I plan to add in the future as I continue learning more about media queries and mobile-friendly layouts.
I also learned the importance of well organized CSS. As the project grew, handling styles became more complex, reinforcing why a clean and consistent structure is crucial in larger projects something I plan to apply when reorganizing my CSS in the future.
Overall, this project is a learning experience, and I plan to keep improving it as my skills develop.
